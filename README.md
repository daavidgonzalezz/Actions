# GitHub Actions Quickstart

Este repositorio incluye el workflow de ejemplo del quickstart oficial de GitHub Actions.

## Workflow incluido

- Archivo: `.github/workflows/github-actions-demo.yml`
- Nombre del workflow: `GitHub Actions Demo`
- Trigger: `push`
- Job: `Explore-GitHub-Actions`

## Cómo probarlo

1. Haz commit y push de estos archivos a un repositorio en GitHub.
2. Ve a la pestaña **Actions** del repositorio.
3. Abre la ejecución de **GitHub Actions Demo** para ver los logs de cada paso.

Referencia: https://docs.github.com/en/actions/get-started/quickstart
